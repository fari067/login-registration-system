# Login & Registration System

A simple Python-based Login and Registration System.

## Features

- User Registration
- User Login
- View All Users
- Logout
- Change Password
- Delete Account
- Exit Program

## Concepts Used

- Lists
- Dictionaries
- Loops
- Conditional Statements
- User Authentication

## How to Run

```bash
python main.py
```

## Example Features

### Register
Create a new account using a username and password.

### Login
Login with a registered username and password.

### Change Password
Update your account password after logging in.

### Delete Account
Delete your account permanently.
